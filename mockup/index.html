The hybrid principle
Brand wraps. Function never bends.

Anthropic's warm cream + Poppins + Lora set the tone — calm, considered, human, not a pushy real-estate brochure. But where a user is being warned that they might lose ₹1,00,000, we keep the colours universal: green means safe, amber means caution, red means risky.

Every other surface — buttons, links, headers, dividers — uses the Anthropic Orange as the primary accent. Numbers stay in JetBrains Mono so they read as data, not marketing.
